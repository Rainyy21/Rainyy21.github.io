## 📦 How to Run Locally

To get this project running on your computer, follow these steps:

    Clone the repository:
        git clone https://github.com/Rainyy21/Rainyy21.github.io

    Navigate into the folder:
        cd Rainyy21.github.io

    Open the site:
    Simply double-click index.html or use the "Live Server" extension in VS Code.
